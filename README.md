## Excel & Power BI Project - Superstore Analysis
## Project Overview
This repository contains a project where we analyze a sample Superstore dataset using **Excel** and **Power BI**.  
The main goal is to demonstrate **data visualization, data analysis, and insights extraction** using these tools.

## Files in this Repository
1. **sample_-_superstore.xls** - Excel file containing raw dataset and basic analysis (Pivot Tables, Charts).  
2. **internshipom.pbix** - Power BI file with interactive dashboards and visualizations.  

## How to Open
- **Excel File (.xlsx)**: Open with Microsoft Excel 2016 or above.  
- **Power BI File (.pbix)**: Open with **Power BI Desktop**.  
  - Download Power BI Desktop here: [https://powerbi.microsoft.com/desktop](https://powerbi.microsoft.com/desktop)

## Features
- Data cleaning and preparation in Excel  
- Pivot tables and charts for insights  
- Interactive dashboards in Power BI  
- Analysis of sales, profit, and customer trends  

## Usage
1. Open the Excel file to explore raw data and basic charts.  
2. Open the Power BI file to interact with dashboards and filters.  
3. You can customize visualizations based on your analysis requirements.
